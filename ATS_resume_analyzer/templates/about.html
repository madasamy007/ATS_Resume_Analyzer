About This System

Overview

The High-Scale Semantic ATS Resume Screening System is an advanced AI-powered solution designed to help HR departments efficiently process large volumes of resumes. Unlike traditional keyword-based ATS systems, this platform uses semantic analysis to understand the meaning and context of resumes and job descriptions.

Technology Stack

  • Backend: FastAPI - Modern, fast web framework for building APIs
  • ML/AI: SentenceTransformers (all-MiniLM-L6-v2) for semantic embeddings
  • ML Model: Logistic Regression for fit prediction
  • Text Extraction: pdfplumber (PDF) and python-docx (DOCX)
  • Database: SQLite for persistent storage
  • Frontend: Tailwind CSS with responsive design

Key Features

  • Bulk processing of up to 1,000 resumes simultaneously
  • Semantic similarity scoring (0-100) using cosine similarity
  • Machine learning-based fit prediction
  • Automatic email extraction and sending
  • Real-time progress tracking
  • Comprehensive dashboard with analytics
  • Excel export functionality

How It Works

The system processes resumes in batches of 50 to ensure efficient handling of large volumes. Each resume is converted into a semantic embedding vector using SentenceTransformers. The job description is also embedded once and cached for reuse. Cosine similarity between resume and job description embeddings provides the semantic score. A trained Logistic Regression model predicts whether a candidate is a fit. Based on these scores, automated emails are sent to candidates.
